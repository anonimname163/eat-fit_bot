# Чек-лист приёмки рефактора Eat&fit (паритет FR-*)

Ручная проверка, что новый стек (NestJS API + бот + Next.js Mini App) закрывает требования
PRD/Architecture. Отмечай `[x]` по мере прохождения. Данные демо/одноразовы — можно ломать.

**Условные обозначения:** 🤖 — в Telegram-боте · 📱 — в Mini App · 🌐 — в браузере · 🔌 — API (curl/Swagger).

> **Авто-прогон API/домена (2026-06-07): 35/35 PASS** — включая веб-регистрацию/вход (телефон+пароль),
> демо-меню и заказ веб-пользователем. Пункты с `✓🔌` подтверждены программно (харнес: forged initData →
> JWT + веб-register/login → батарея запросов), даже если строка помечена 🤖/📱 — там проверялась
> серверная логика. Непомеченные `[ ]` требуют визуального/ручного подтверждения в Telegram/Mini App
> (рендер UI, доставка уведомлений, фото, бот-клавиатуры, отчёты, темы).
